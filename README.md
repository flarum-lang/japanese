# Japanese Language Pack for Flarum
[![CI](https://github.com/flarum-lang/japanese/workflows/CI/badge.svg)](https://github.com/flarum-lang/japanese/actions?query=workflow%3ACI)<br>
Japanese language pack to localize the Flarum.

## :inbox_tray: гӮӨгғігӮ№гғҲгғјгғ«ж–№жі• | Installation
1. FlarumгҒҢгӮӨгғігӮ№гғҲгғјгғ«гҒ•гӮҢгҒҰгҒ„гӮӢгғҮгӮЈгғ¬гӮҜгғҲгғӘгҒ§д»ҘдёӢгҒ®гӮігғһгғігғүгӮ’е®ҹиЎҢгҒ—гҒҰгҒҸгҒ гҒ•гҒ„гҖӮ<br>
Run the following command in the location where Flarum is installed.
```
composer require flarum-lang/japanese
```

2. з®ЎзҗҶз”»йқўгҒ®жӢЎејөж©ҹиғҪгғҡгғјгӮёгҒӢгӮүгҖҒ`` Japanese ``гӮ’жңүеҠ№еҢ–гҒ—гҒҫгҒҷгҖӮ<br>
Go to Extensions page of the admin interface and enable ``Japanese``.

3. з®ЎзҗҶз”»йқўгҒ®еҹәжң¬гғҡгғјгӮёгҒӢгӮүгғҮгғ•гӮ©гғ«гғҲгҒ®иЁҖиӘһгӮ’``Japanese``гҒ«еӨүжӣҙгҒ—гҒҫгҒҷгҖӮ<br>
Go to Basics page of the admin interface and change default language to ``Japanese``.

## :recycle: гӮўгғғгғ—гғҮгғјгғҲгҒЁеүҠйҷӨж–№жі• | Update and Uninstall
FlarumгҒҢгӮӨгғігӮ№гғҲгғјгғ«гҒ•гӮҢгҒҰгҒ„гӮӢгғҮгӮЈгғ¬гӮҜгғҲгғӘгҒ§д»ҘдёӢгҒ®гӮігғһгғігғүгӮ’е®ҹиЎҢгҒ—гҒҰгҒҸгҒ гҒ•гҒ„гҖӮ<br>
Run the following command in the location where Flarum is installed.

- гӮўгғғгғ—гғҮгғјгғҲ | Update
```
composer update flarum-lang/japanese
```
